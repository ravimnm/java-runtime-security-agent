# JRSA - Java Runtime Security Agent JRSA (Java Runtime Security Agent) is an in-process JVM security agent that monitors, analyzes, and reports high-risk runtime activities occurring inside Java applications. Unlike traditional security mechanisms that rely on static analysis or network inspection, JRSA focuses on **runtime reality** by observing sensitive JVM operations as they are executed. JRSA uses the Java Instrumentation API and ByteBuddy to intercept critical runtime behaviors, perform behavioral analysis, and generate security telemetry in real time. --- ## Features * Runtime bytecode instrumentation using Java Instrumentation API. * Dynamic interception of sensitive JVM operations. * Real-time monitoring of process execution activities. * Reflection abuse detection. * Rule-based behavioral analysis and risk scoring. * Real-time event streaming to SMTAP. * Tamper-evident centralized security logging. * Pluggable architecture for future detection modules. --- ## Current Detection Capabilities ### Process Execution Monitoring JRSA intercepts: ```java Runtime.getRuntime().exec(...) ProcessBuilder.start() ``` Detects: * Shell spawning * PowerShell execution * Encoded PowerShell commands * Command execution from protected applications Example: ```text powershell.exe -enc SQBtACAAbQBhAGwAaQBjAGkAbwB1AHMA ``` --- ### Reflection Usage Monitoring JRSA monitors reflection-based execution patterns: ```java Class.forName(...) Method.invoke(...) ``` Detects: * Dynamic class loading * Reflection abuse * Suspicious runtime invocation patterns --- ## Technology Stack * Java 17 * Java Instrumentation API * ByteBuddy * Jackson * Maven --- ## Performance Metrics Benchmarks were performed using SMTAP as the protected JVM application. | Metric                        | Result         | | ----------------------------- | -------------- | | Event Throughput              | ~70 events/sec | | Startup Overhead              | ~9.7%          | | Memory Overhead               | ~23.5 MB       | | Runtime Monitoring Categories | 2+             | | Telemetry Delivery            | Real-time      | --- ## Example Usage Attach JRSA to any JVM application: ```bash java -javaagent:jrsa.jar -jar application.jar ``` Example: ```bash java ^ -javaagent:jrsa.jar ^ -jar smtap.jar ``` --- ## Example Security Event ```text Event ID: 123 Timestamp: 2026-06-24 17:10:00 Observed Activity: Process Execution Command: powershell.exe -enc SQBtACAAbQBhAGwAaQBjAGkAbwB1AHMA Risk Score: 90/100 Reason: -
